## Cách cập nhật lên GitHub
1. Giải nén ZIP.
2. Lấy file `index.html`.
3. Vào repository GitHub cũ.
4. Upload file `index.html` mới để thay file cũ.
5. Commit changes.
6. Chờ 1-3 phút rồi mở lại GitHub Pages.
